## 2. Data Architecture (Task 1)

### 2.1 Corpus and station selection

| Field | Detail |
| :--- | :--- |
| **Archive** | UCI ML Repository mirror — PRSA Beijing Multi-Site Air Quality (2013–2017) |
| **Academic framing** | Official corpus of **Zhang et al. (2017)**, *Atmospheric Environment* — see `README.md` §1.3 |
| **Bundled stations** | 12 CSV files under `PRSA_Data/PRSA_Data_20130301-20170228/` |
| **Primary site** | **Aotizhongxin** — $N = 35{,}064$ hourly records |
| **Features** | PM2.5, PM10, SO₂, NO₂, CO, O₃, meteorology, one-hot `wd` |

### 2.2 Preprocessing contract (preserved by Tasks 2–4)

1. Continuous `datetime` index; drop non-informative ID columns.  
2. Missing values: linear interpolation on numeric columns → `ffill` → `bfill`.  
3. One-hot `wd` (`drop_first=True`).  
4. **PM2.5** at column index 0.  
5. Chronological 70% / 15% / 15% split.  
6. `MinMaxScaler` fit **only on train**.  
7. Windows: $\mathbf{X} \in \mathbb{R}^{24 \times D} \rightarrow y_{T+1}$.

**Team extensions (architecture unchanged for Task 1):** `test_target_hours` for attention analytics; pandas ≥2.1 numeric-only `interpolate`; DAE training noise on auxiliary channels (Task 2).

### 2.3 Window shapes (representative)

| Split | Approx. hours | Windows (24h lag) |
| :--- | :---: | :---: |
| Train | 24,544 | ~24,520 |
| Validation | 5,260 | ~5,236 |
| Test | 5,260 | ~5,236 |

### 4.4 Key findings (current pipeline)

- **BiLSTM + CNN + residual + attention** yield $R^2 > 0.90$ on clean data for all variants.  
- Under **noisy auxiliary sensors**, Model **A** stays at $R^2 = 0.9170$ while Model **B** falls to $0.8692$ ($\Delta R^2 = -0.0467$ for B).  
- Model **D** collapses most under noise ($\Delta R^2 = -0.0564$), showing the full stack is required for deployment realism.  
- Detailed narrative: `outputs/dae_noise_robustness_report.md`, figures in `outputs/figure_catalog.md`.
